# 🎵 Spotify DER

## 📖 Sobre o Projeto

Este repositório apresenta um Modelo Entidade-Relacionamento (DER) Conceitual inspirado na estrutura de dados da plataforma Spotify.

O projeto foi desenvolvido para aplicar conceitos de modelagem de banco de dados, incluindo entidades, atributos, relacionamentos, cardinalidades, entidades associativas e normalização.

---

## 🎯 Objetivos

- Modelar um sistema de streaming de música.
- Aplicar conceitos de MER e DER.
- Identificar entidades e relacionamentos.
- Representar regras de negócio do Spotify.
- Praticar modelagem conceitual de banco de dados.

---

## 🚀 Objetivo Acadêmico

Este projeto foi desenvolvido para a disciplina de Banco de Dados, com o objetivo de representar de forma conceitual a estrutura de um sistema de streaming musical semelhante ao Spotify.

O modelo contempla as principais funcionalidades da plataforma, incluindo gerenciamento de usuários, playlists, músicas, artistas, álbuns, podcasts, histórico de reprodução e bibliotecas pessoais.
